Ajuste do modelo para Tabaco + Cardiaco

Neste caso, foram ajustados dois modelos com a relao aditiva entre as variveis \(\text{cardiaco + tabaco}\), so eles :

  • Modelo de chances proporcionais (MCP), que assume que os efeitos das covariveis so constantes entre os diferentes pontos de corte da escala ordinal.

  • Modelo de chances no proporcionais (MCNP), que relaxa essa suposio, permitindo que o efeito das covariveis varie entre os nveis da resposta ordinal.
